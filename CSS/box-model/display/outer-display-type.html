Understanding Outer Display Types

I am a block-level element (div). I take up the full width of the container and start on a new line.
I am an inline element (span). I take only the space I need and stay on the same line. Another inline element. I sit next to the previous one.
I am an inline-block element (div). I sit inline, but I respect block-level dimensions.
Another inline-block element. I can sit next to the previous one.
